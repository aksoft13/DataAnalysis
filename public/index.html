myDataAnalysis - 앱 소개
멀티소스 데이터 수집 및 SNA 기반 비교 분석 플랫폼

개발 목적

대학원 수업(데이터 수집 및 분석 프로그래밍) 조별 과제용으로 제작된 웹 기반 분석 도구입니다. 뉴스와 유튜브에서 특정 이벤트에 대한 대중 반응을 수집하고, 키워드 분석 및 SNA 네트워크 시각화를 통해 담론 구조를 비교 분석합니다.

기술 스택

  • 프론트엔드: HTML, CSS, JavaScript, D3.js (SNA 시각화), Chart.js (차트)
  • 백엔드: Node.js, Express, Axios, XLSX
  • 형태소 분석: Python 3, kiwipiepy (C++ 기반 한국어 NLP)
  • 데이터 소스: Google News RSS, YouTube Data API v3
  • AI 연동: 복사-붙여넣기 방식 (API 키 불필요)

주요 기능

  • 멀티소스 데이터 수집: Google News + YouTube (영상 메타데이터 + 댓글)
  • 그룹 비교 분석: 최대 6개 그룹별 키워드·SNA·참여도 비교
  • 소스별 분리 분석: 뉴스 vs 유튜브 댓글 키워드 개별 분석
  • 감성 분석: 키워드 사전 기반 긍정/부정/중립 분류 + 좋아요 가중 공감도
  • SNA 네트워크: 키워드 동시출현 네트워크, Louvain 커뮤니티 탐지, PageRank, Eigenvector
  • EDA 탐색: 산점도, 히스토그램, 박스플롯, 히트맵, 페어플롯
  • AI 프롬프트 자동 생성: 분석 결과 기반 보고서 작성용 프롬프트
  • 내보내기: Excel(XLSX), CSV, JSON 형식 지원

업데이트 이력

  • v1.0 (2026-04): 초기 버전 - 뉴스/블로그/유튜브 수집 + 키워드 분석 + SNA
  • v1.1 (2026-04): EDA 차트 추가 (산점도, 히트맵, 페어플롯)
  • v1.2 (2026-05): Google News 검색 지원 (기간 필터링 가능)
  • v1.3 (2026-05): 소스별 분리 분석 (뉴스 vs 유튜브 키워드 분리)
  • v1.4 (2026-05): 감성 분석 추가 (키워드 사전 + 좋아요 가중 공감도)

시스템 아키텍처

브라우저 (HTML/JS/D3.js/Chart.js)
    |  HTTP localhost:3007
Node.js + Express 서버
    |-- Google News RSS (뉴스 수집)
    |-- YouTube Data API v3 (영상 + 댓글 수집)
    |-- Python kiwipiepy (형태소 분석, child_process)
    |-- XLSX/CSV (데이터 내보내기)
    +-- JSON (작업 저장/불러오기)

분석 파이프라인

[데이터 수집] → [텍스트 정제] → [형태소 분석(kiwipiepy)]
    → [키워드 빈도 집계] → [동시출현 매트릭스]
    → [SNA 네트워크 구축] → [커뮤니티 탐지]
    → [감성 분석] → [시각화 + 내보내기]
