1. 문제 인식 & 동기

실무에서의 데이터 분석 현실

이 과목의 핵심은 데이터를 수집하고 분석하여 각자의 상황에 맞게 처리하는 것입니다. 그런데 실무에서는 이 과정이 반복 작업의 연속입니다.

  • 매번 반복되는 작업: 여러 사이트에서 수집 → 포맷 통일 → 분석 → 시각화 → 해석
  • 채널별 별도 도구: 뉴스는 빅카인즈, 텍스트는 텍스톰, 유튜브는 별도 크롤러
  • 비교 분석의 어려움: 시점별, 채널별 비교를 하려면 같은 작업을 여러 번 반복

해결 방안

이 반복 작업을 앱으로 자동화하면 불필요한 시간 낭비를 줄이고, 누구나 같은 품질의 분석을 빠르게 수행할 수 있습니다.

  • 하나의 앱에서 End-to-End 완결: 수집 → 분석 → 시각화 → AI 해석까지 원스톱
  • 주제만 바꾸면 바로 재사용: 앱 구조가 동일하므로 어떤 주제든 같은 워크플로우로 분석 가능

2. 프로젝트 개요

myDataAnalysis는 하나의 연구 질문에서 출발하여, 뉴스 + 블로그 + 유튜브 데이터를 동시에 수집하고, 비교 그룹별 SNA 네트워크 분석 + EDA 시각화 + AI 해석 프롬프트까지 하나의 앱에서 End-to-End로 완결하는 통합 분석 플랫폼입니다.

기존 방식myDataAnalysis효과
채널별 별도 도구 사용뉴스 + 블로그 + 유튜브 한 곳에서 수집작업 시간 단축, 포맷 통일
단일 시점 분석비교 그룹별 나란히 분석변화 추이, 차이점 즉시 파악
수집 따로, 분석 따로, 해석 따로수집 → 형태소 → SNA → EDA → AI 원스톱워크플로우 일원화

실무 활용 시나리오

  • PR팀: 브랜드 위기 전/중/후 여론 프레임 변화 추적
  • 마케팅팀: 경쟁 브랜드 간 미디어 포지셔닝 비교
  • 리서치팀: 사회 이슈에 대한 채널별 담론 분석
  • 기획팀: 신제품 출시 전후 소비자 인식 변화 모니터링

3. 분석 플로우

연구 질문부터 인사이트까지, 5단계 워크플로우:

1연구 질문 입력
자유 텍스트로 분석 목적 기술 + 작업 히스토리 저장/로드 NEW
▼
2비교 그룹 설정
그룹별 검색어, 기간, 소스, 수집 건수(최대 1,000건) 직접 설정
▼
3멀티소스 데이터 수집
네이버 뉴스/블로그 + 구글 뉴스 + 유튜브(영상 메타 + 댓글) — 그룹별 CSV 다운로드 NEW
▼
4분석 결과
형태소 분석 + 키워드 비교 + SNA 네트워크(Louvain, PageRank, Eigenvector) + EDA 시각화 NEW
▼
5AI 프롬프트 + 보고서
7가지 관점 프롬프트 → ChatGPT/Claude/Gemini 복사 → 보고서 생성(HTML/MD/PDF) NEW

4. 데이터 소스

네이버 뉴스

API (최대 1,000건) + 웹 스크래핑 (날짜 검색)

네이버 블로그

API (최대 1,000건) + 구글 RSS 경유

구글 뉴스

RSS 기반, API 키 불필요. 날짜 분할 대량 수집

유튜브

YouTube Data API v3. 영상 메타 + 댓글 수집

소스API 키날짜 검색최대 건수수집 항목
네이버 뉴스 (API)필요불가1,000건제목, 본문 요약, 언론사, 날짜
네이버 뉴스 (웹)불필요가능~100건제목, 본문 요약, 언론사, 날짜
구글 뉴스 (RSS)불필요가능1,000건+제목, 요약, 출처, 날짜
유튜브 (API)필요가능설정 가능영상 제목, 조회수, 좋아요, 댓글

6. EDA 시각화

산점도 (Scatter Plot)

댓글 수 vs 좋아요 — 참여도 상관관계
영상 길이 vs 조회수 — 최적 영상 길이 패턴

히스토그램 + 박스플롯

그룹별 조회수 분포 비교
중앙값, 사분위, 이상치 한눈에 파악

히트맵 (Heatmap)

요일 × 시간대별 영상 업로드 분포
요일 × 시간대별 댓글 작성 분포

페어플롯 (Pair Plot)

조회수·좋아요·댓글수·영상길이 4개 지표의 모든 조합 상관관계를 4×4 매트릭스로 비교. 그룹별 필터링 가능

시계열 차트

뉴스 기사 수 시계열 추이 (그룹별 비교)
유튜브 영상 게시일 분포
그룹별 총 조회수 / 좋아요+댓글 비교

8. AI 프롬프트 연동

AI API 없이, 복사-붙여넣기로 AI를 활용합니다.

7가지 해석 프롬프트 (자동 생성)

관점설명
인사이트핵심 패턴, 모듈성 기반 해석
학술적의제설정, 프레이밍 이론
사회적사회 구조적 맥락
문화적문화 코드, 담론 해석
기술적기술 트렌드, 산업 변화
비즈니스마케팅/PR 전략 도출
통합 보고서종합 보고서 (docx/html/pdf)

통합 보고서 자동 구성

  • 표지 + 목차 (점선 리더 + 페이지 번호)
  • 서론, 데이터 수집 개요, 키워드 분석, SNA 네트워크 분석
  • 그룹별 비교 (5가지 관점), 결론 및 제언, 참고문헌
  • 14개 그래프 삽입 위치 자동 표시
  • ChatGPT / Claude / Gemini 원클릭 바로가기 버튼

9. 추가 기능

작업 히스토리

연구 질문 + 그룹 설정 + 수집 데이터 + 분석 결과를 JSON으로 저장. 클릭 한 번으로 즉시 복원

데이터 내보내기

그룹별 CSV 다운로드. 전체 ZIP(5개 CSV). SNA XLSX/CSV (Gephi 호환)

보고서 생성

AI 해석 결과 붙여넣기 → 보고서 자동 구성. HTML / Markdown / PDF 출력

불용어 관리

기본 70+개. 실시간 제외 버튼. 수동 추가/삭제

첫 실행 설치 가이드

Python / kiwipiepy 자동 감지. 미설치 시 원클릭 설치. 설치 없이 정규식 모드 동작

독립 실행파일 배포

Mac (Apple Silicon / Intel) + Windows. 더블클릭으로 실행, 브라우저 자동 열림
